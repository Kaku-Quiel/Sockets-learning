📖¿Qué es un Socket?

La base de toda comunicación en red

Imagina un socket como un enchufe virtual entre dos programas. Uno es el servidor (espera conexiones) y otro el cliente (inicia la conexión). A través de ese "enchufe" se envían y reciben datos, como si hablaras por un walkie-talkie digital.

En Python, todo esto se maneja con la librería incorporada socket. No necesitas instalar nada extra.

💡 Dato clave: Usaremos TCP (protocolo confiable, como una llamada telefónica) en lugar de UDP (rápido pero sin garantía, como un grito). TCP asegura que los datos lleguen completos y en orden.

1️⃣Ejercicio 1 — Crear un socket y entenderlo

Tu primer "enchufe" virtual

01

Objetivo

Crear un objeto socket, imprimir su información y cerrarlo. Sin conexión todavía, solo familiarizarte con la estructura.

  • Aprende a importar socket
  • Entiende los parámetros: AF_INET (IPv4) y SOCK_STREAM (TCP)
  • Obtén el nombre del host local
📄 ejercicio1_info_socket.py 
import socket

# Crear un socket TCP/IPv4
mi_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# Obtener el nombre del host local
hostname = socket.gethostname()
ip_local = socket.gethostbyname(hostname)

print(f"✅ Socket creado: {mi_socket}")
print(f"🖥️  Hostname: {hostname}")
print(f"📍 IP local: {ip_local}")
print(f"📦 Familia: IPv4 (AF_INET)")
print(f"🔗 Tipo: TCP (SOCK_STREAM)")

# Cerrar el socket
mi_socket.close()
print("🔒 Socket cerrado.")
▼ Salida esperada
✅ Socket creado: <socket.socket fd=3, family=AddressFamily.AF_INET, type=SocketKind.SOCK_STREAM, proto=0> 🖥️ Hostname: mi-pc 📍 IP local: 192.168.1.10 📦 Familia: IPv4 (AF_INET) 🔗 Tipo: TCP (SOCK_STREAM) 🔒 Socket cerrado.
✅ ¡Listo! Acabas de crear tu primer socket. No hizo nada en red todavía, pero ya entiendes sus partes. Ejecútalo en tu terminal con python ejercicio1_info_socket.py.

2️⃣Ejercicio 2 — Servidor TCP que saluda

El que espera y responde

02

Objetivo

Crear un servidor que escuche en localhost:5000, acepte una conexión y envíe un mensaje de bienvenida al cliente que se conecte.

  • Usa bind() para asignar dirección y puerto
  • Usa listen() para ponerse a la escucha
  • Usa accept() para recibir la conexión
  • Envía datos con send()
📄 ejercicio2_servidor_saludo.py 
import socket

# Configuración
HOST = '127.0.0.1'    # localhost = solo conexiones de esta máquina
PORT = 5000         # Puerto (elegir uno mayor a 1024 para evitar permisos)

# Crear socket servidor
servidor = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
servidor.bind((HOST, PORT))
servidor.listen(1)  # Máximo 1 cliente en cola de espera

print(f"🚀 Servidor escuchando en {HOST}:{PORT}...")
print("⏳ Esperando que un cliente se conecte...\n")

# Aceptar conexión entrante
conexion, direccion = servidor.accept()
print(f"✅ ¡Cliente conectado desde {direccion}!")

# Enviar mensaje de bienvenida
mensaje = "¡Hola, cliente! Bienvenido al servidor 🎉\n"
conexion.send(mensaje.encode('utf-8'))  # Convertir string a bytes

print("📤 Mensaje enviado. Cerrando conexión...")
conexion.close()
servidor.close()
print("🔒 Servidor cerrado.")
⚠️ Importante: Este servidor se cierra después de atender a un solo cliente. En ejercicios posteriores lo mejoraremos. Por ahora, para probarlo necesitas el cliente del Ejercicio 3.

3️⃣Ejercicio 3 — Cliente TCP que recibe el saludo

El que inicia la conversación

03

Objetivo

Crear un cliente que se conecte al servidor del Ejercicio 2, reciba el mensaje y lo muestre en pantalla.

  • Usa connect() para establecer conexión
  • Usa recv() para recibir datos
  • Decodifica los bytes recibidos con .decode('utf-8')
📄 ejercicio3_cliente_saludo.py 
import socket

HOST = '127.0.0.1'  # El mismo que el servidor
PORT = 5000

# Crear socket cliente
cliente = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

print(f"🔗 Intentando conectar a {HOST}:{PORT}...")
cliente.connect((HOST, PORT))
print("✅ Conectado al servidor!\n")

# Recibir datos (buffer de 1024 bytes)
datos = cliente.recv(1024)
mensaje = datos.decode('utf-8')

print("📥 Mensaje del servidor:")
print("─" * 35)
print(mensaje)
print("─" * 35)

cliente.close()
print("🔒 Conexión cerrada.")
▼ Cómo probarlo (abre 2 terminales)
Terminal 1: python ejercicio2_servidor_saludo.py Terminal 2: python ejercicio3_cliente_saludo.py

4️⃣Ejercicio 4 — Chat simple (ida y vuelta)

Ambos hablan, ambos escuchan

04

Objetivo

Crear un chat donde servidor y cliente se turnan para enviar mensajes hasta que uno escriba "chao".

  • Usa un bucle while True para mantener la conversación
  • Alterna entre enviar y recibir en cada lado
  • Maneja la desconexión con "chao"
📄 ejercicio4_chat_servidor.py 
import socket

HOST = '127.0.0.1'
PORT = 5001

servidor = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
servidor.bind((HOST, PORT))
servidor.listen(1)
print(f"💬 Chat - Esperando conexión en {HOST}:{PORT}...")

conexion, direccion = servidor.accept()
print(f"✅ Conectado a {direccion}\n")
print("💡 Escribe 'chao' para terminar.\n")

while True:
    # 1. Recibir mensaje del cliente
    datos = conexion.recv(1024)
    if not datos:
        break
    msg_cliente = datos.decode('utf-8').strip()
    print(f"👤 Cliente: {msg_cliente}")
    if msg_cliente.lower() == 'chao':
        print("👋 El cliente se despidió.")
        break

    # 2. Enviar respuesta
    respuesta = input("👤 Tú (servidor): ")
    conexion.send(respuesta.encode('utf-8'))
    if respuesta.lower() == 'chao':
        print("👋 Te despediste. Fin del chat.")
        break

conexion.close()
servidor.close()

5️⃣Ejercicio 5 — Servidor Multi-Cliente con hilos

Atendiendo a varios al mismo tiempo

05

Objetivo

Crear un servidor que pueda atender a múltiples clientes simultáneamente usando threading. Cada cliente recibe un saludo personalizado.

  • Cada conexión se maneja en un hilo separado
  • El servidor nunca se bloquea para otros mientras atiende a uno
  • Ideal para entender cómo funcionan los chats reales
📄 ejercicio5_servidor_multi.py 
import socket
import threading

HOST = '127.0.0.1'
PORT = 5002

def manejar_cliente(conexion, direccion, num_cliente):
    """Función que ejecuta cada hilo para un cliente."""
    print(f"🟢 [Hilo {num_cliente}] Cliente conectado: {direccion}")
    saludo = f"¡Hola Cliente #{num_cliente}! Eres atendido por un hilo dedicado.\n"
    conexion.send(saludo.encode('utf-8'))
    conexion.close()
    print(f"🔴 [Hilo {num_cliente}] Cliente {direccion} desconectado.")

servidor = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
servidor.bind((HOST, PORT))
servidor.listen(5)
print(f"🚀 Servidor Multi-Cliente en {HOST}:{PORT}")
print("⏳ Esperando conexiones (abre varios clientes)...\n")

contador = 0
try:
    while True:
        conexion, direccion = servidor.accept()
        contador += 1
        hilo = threading.Thread(
            target=manejar_cliente,
            args=(conexion, direccion, contador),
            daemon=True
        )
        hilo.start()
except KeyboardInterrupt:
    print("\n⛔ Servidor detenido.")
finally:
    servidor.close()

🚀 PROYECTO FINAL

💬Sala de Chat Multiusuario

Combina todo lo aprendido en un solo proyecto funcional

Vas a construir un chat room donde múltiples clientes pueden conectarse, elegir un alias, enviar mensajes públicos a toda la sala, y ver quién entra o sale. El servidor retransmite cada mensaje a todos los demás (broadcast).

📋 Requisitos del proyecto

  • ✅ Servidor con threading para manejar múltiples clientes
  • ✅ Cada cliente elige un alias al conectarse
  • ✅ Cuando alguien envía un mensaje, todos lo ven (broadcast)
  • ✅ Notificación cuando un usuario entra o sale de la sala
  • ✅ El cliente usa dos hilos: uno para leer del servidor y otro para escribir
  • ✅ Comando /salir para desconectarse limpiamente

🏗️ Arquitectura

ComponenteResponsabilidadTécnica usada
ServidorAcepta conexiones, guarda clientes activos, retransmite mensajessocket + threading
Cliente (hilo lector)Escucha constantemente mensajes del servidor y los imprimethreading.Thread
Cliente (hilo escritor)Lee desde el teclado y envía al servidorHilo principal / input()
BroadcastEl servidor recorre todos los clientes y reenvía el mensajeBucle for sobre lista de sockets

📄 Código del Servidor

📄 proyecto_chat_servidor.py 
import socket
import threading

HOST = '127.0.0.1'
PORT = 5555

# Lista compartida: guarda tuplas (socket, alias)
clientes = []
lock = threading.Lock()  # Para acceso seguro a la lista

def broadcast(mensaje, remitente_socket=None):
    """Envía mensaje a TODOS los clientes excepto al remitente."""
    with lock:
        for cliente_sock, _ in clientes:
            if cliente_sock != remitente_socket:
                try:
                    cliente_sock.send(mensaje.encode('utf-8'))
                except:
                    pass  # Cliente ya desconectado, se ignorará

def manejar_cliente(cliente_sock, direccion):
    """Hilo dedicado a cada cliente conectado."""
    try:
        # Pedir alias
        cliente_sock.send("👤 Ingresa tu alias: ".encode('utf-8'))
        alias = cliente_sock.recv(1024).decode('utf-8').strip()
        if not alias:
            alias = f"User_{direccion[1]}"

        # Registrar cliente
        with lock:
            clientes.append((cliente_sock, alias))
        print(f"🟢 [{alias}] se unió desde {direccion}")
        broadcast(f"📢 [{alias}] ha entrado a la sala. ¡Saluden!\n", cliente_sock)
        cliente_sock.send(f"✅ Bienvenido a la sala, {alias}!\n".encode('utf-8'))
        cliente_sock.send("💡 Escribe /salir para abandonar la sala.\n".encode('utf-8'))

        # Bucle de recepción de mensajes
        while True:
            datos = cliente_sock.recv(1024)
            if not datos:
                break
            msg = datos.decode('utf-8').strip()
            if msg.lower() == '/salir':
                break
            print(f"💬 [{alias}]: {msg}")
            broadcast(f"[{alias}]: {msg}\n", cliente_sock)
    except Exception as e:
        print(f"⚠️ Error con [{alias}]: {e}")
    finally:
        # Desconectar limpiamente
        with lock:
            for item in clientes:
                if item[0] == cliente_sock:
                    clientes.remove(item)
                    break
        print(f"🔴 [{alias}] abandonó la sala.")
        broadcast(f"📢 [{alias}] ha salido de la sala.\n")
        cliente_sock.close()

# ── Iniciar servidor ──
servidor = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
servidor.bind((HOST, PORT))
servidor.listen(10)
print(f"🚀 Sala de Chat activa en {HOST}:{PORT}")
print("⏳ Esperando usuarios...\n")

try:
    while True:
        cliente_sock, direccion = servidor.accept()
        hilo = threading.Thread(target=manejar_cliente, args=(cliente_sock, direccion), daemon=True)
        hilo.start()
except KeyboardInterrupt:
    print("\n⛔ Servidor detenido.")
finally:
    servidor.close()

📄 Código del Cliente

📄 proyecto_chat_cliente.py 
import socket
import threading

HOST = '127.0.0.1'
PORT = 5555

def recibir_mensajes(sock):
    """Hilo: escucha constantemente mensajes del servidor."""
    while True:
        try:
            datos = sock.recv(1024)
            if not datos:
                print("\n🔌 Conexión cerrada por el servidor.")
                break
            print(datos.decode('utf-8'), end='')
        except:
            break

# ── Conectar ──
cliente = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
cliente.connect((HOST, PORT))

# Iniciar hilo receptor
hilo_receptor = threading.Thread(target=recibir_mensajes, args=(cliente,), daemon=True)
hilo_receptor.start()

# Bucle principal: enviar mensajes
try:
    while True:
        mensaje = input()
        if mensaje.strip().lower() == '/salir':
            cliente.send(mensaje.encode('utf-8'))
            break
        cliente.send(mensaje.encode('utf-8'))
except KeyboardInterrupt:
    pass
finally:
    cliente.close()
    print("\n👋 Desconectado.")
