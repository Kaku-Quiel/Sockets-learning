2. CERRAR UN ARCHIVO

Forma manual (no recomendada):

archivo = open("archivo.txt", "r")
contenido = archivo.read()
archivo.close() # Obligatorio

Forma automática con with (recomendada):

with open("archivo.txt", "r") as archivo:
    contenido = archivo.read()
# Aquí se cierra automáticamente, incluso si hay error

5. MANEJAR EL CURSOR

MétodoDescripciónEjemplo
.tell()Devuelve la posición actual del cursor (en bytes)pos = archivo.tell()
.seek(posición)Mueve el cursor a una posición específicaarchivo.seek(0) (al principio)
with open("archivo.txt", "r+") as f:
    print(f.tell()) # 0
    contenido = f.read()
    print(f.tell()) # Número de bytes leídos
    f.seek(0) # Vuelve al principio
    f.write("Nuevo texto al inicio")

7. TRABAJAR CON JSON

8. RUTAS DE ARCHIVOS

Relativas:

"archivo.txt" # Misma carpeta
"carpeta/archivo.txt" # Subcarpeta
"../carpeta/archivo.txt" # Subir un nivel

Absolutas:

# Windows
"C:/Users/tu_usuario/Documentos/archivo.txt"
"C:\\Users\\tu_usuario\\Documentos\\archivo.txt" # Linux/Mac
"/home/tu_usuario/Documentos/archivo.txt"

Con pathlib (recomendado):

from pathlib import Path # Obtener carpeta del script carpeta = Path("__file__").parent # Construir ruta ruta = carpeta / "subcarpeta" / "archivo.txt" # Crear carpetas si no existen ruta.parent.mkdir(parents=True, exist_ok=True) # Abrir archivo with open(ruta, "w") as f:
    f.write("Hola")

Con os.path:

import os carpeta = os.path.dirname("__file__")
ruta = os.path.join(carpeta, "subcarpeta", "archivo.txt")
os.makedirs(os.path.dirname(ruta), exist_ok=True)

Ver directorio actual:

import os
print(os.getcwd()) # Current Working Directory

10. CODIFICACIÓN (ENCODING)

Especificar encoding (recomendado siempre):

with open("archivo.txt", "r", encoding="utf-8") as f:
    contenido = f.read()

Encodings comunes:

  • utf-8 – El estándar universal Recomendado
  • latin-1 o iso-8859-1 – Antiguo, para caracteres europeos
  • cp1252 – Windows occidental
# Leer con encoding diferente with open("archivo.txt", "r", encoding="latin-1") as f:
    contenido = f.read()

11. BUENAS PRÁCTICAS

  • Siempre usa with open() – cierra automáticamente.
  • Siempre especifica encoding="utf-8" – evita sorpresas con tildes y ñ.
  • Usa pathlib para rutas portables entre SO.
  • Crea carpetas con exist_ok=True antes de escribir.
  • Usa 'x' en lugar de 'w' si no quieres sobrescribir accidentalmente.
  • Maneja excepciones (FileNotFoundError, PermissionError).
  • Para archivos grandes, itera línea por línea en lugar de .read().
